### Customization parameters
Customize the parameters as per your need and the chat component being used.

| Param         | Type            | Default | Remarks |
| ------------- | --------------- | ------- | -------- |
| account       | `string`         | -       | Supports wallet address or chain agnostic wallet address format|
| pgpPrivatekey | `string`          | -       | Decrypted pgp private key of the user |
| theme         | `IChatTheme`      | -       | [Theme object](/docs/chat/ui-components/Advanced/Theme/),it can be custom theme object, `lightChatTheme` `darkChatTheme` |
| _`children`_  | `React.ReactNode` | -       ||
| env           | `Env`             | -       |API env - 'prod' or 'staging'|
| signer        | `SignerType`          | -       |EthersV5 or Viem Signer|

> Note: Parameters _`in this style`_ are mandatory.
