### Customization parameters

When sending a message, you can customize the following params

| Param         | Type                | Subtype  | Default | Remarks                                                                                                                                                                                              |
| ------------- | ------------------- | -------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| _`recipient`_ | `string`            | -        | -       | [Recipient supports a number of address format](/docs/chat/build/types-of-recipient) including wallet address, chain agnostic wallet address, NFT addresses or even chatid which is useful in groups |
| _`message`_   | `object`            | -        | -       | Configuration for message to be sent                                                                                                                                                                 |
| -             | _`message.type`_    | `Text`   | `Text`  | Type of message content                                                                                                                                                                              |
| -             | _`message.content`_ | `string` | -       | Pass plain text in the message content                                                                                                                                                               |

> Note: Parameters _`in this style`_ are mandatory.
