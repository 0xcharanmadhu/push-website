# Showrunners Scaffold

This section covers Showrunners scaffold which is generic template from Push for creating automated notification triggering backend up and running in no time.

You only need this in case you are designing notification triggers (For example: loan liquidation notification trigger, NFT buy / sell trigger, RSS feed to notification trigger, etc) from scratch and want a super quick way to get started.
