# UI Components

This Section will show you how you can easily integrate Notification UI widgets, SubscriptionManager in your dapp with just few lines of code using our components from `@pushprotocol/uiweb`

_`Note`_ - If you're using Next.js to integrate UI Components, we recommend using `Next.js 13` with the latest [App Router](https://nextjs.org/docs/app).
