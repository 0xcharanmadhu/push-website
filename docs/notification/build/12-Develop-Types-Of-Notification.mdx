Push Protocol allows a bunch of notifications to be sent to the user.

## Broadcast Notification

This type of notification is basically broadcasted or sent to all the subscribers of a channel at once

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>

```typescript
// sends a broadcast notification
const broadcastNotif = await userAlice.channel.send(['*'], {
  notification: { title: 'test', body: 'test' },
})
```

  </TabItem>
</Tabs>

## Targeted Notification

As the name suggests, this type of notification allows channel owners to notify a specific wallet address.

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>

```typescript
// sends a targeted notification
const targetedNotif = await userAlice.channel.send([signer.address], {
  notification: {
    title: 'test',
    body: 'test',
  },
})
```

  </TabItem>
</Tabs>

## Subset Notification

Subset notifications are quite helpful when a particular group of subscribers of a channel is supposed to be notified

<Tabs className="codetabs" groupId="code-examples">
  <TabItem value="js" attributes={{className: "codetab js"}} default>

```typescript
// sends a subset notification
const subsetNotif = await userAlice.channel.send(
  [randomWallet1, randomWallet2, signer.address],
  {
    notification: {
      title: 'test',
      body: 'test',
    },
  }
)
```

  </TabItem>
</Tabs>
