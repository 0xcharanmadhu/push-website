# Message Type - Text
Enables sending text messages to the recipient.

    await PushAPI.chat.send({
      messageObj: {
        content: "Gm gm! It's me... Mario"
      },
      messageType: 'Text', // can be "Text" | "Image" | "File" | "MediaEmbed" | "Meta"
      receiverAddress: 'eip155:0x0F1AAC847B5720DDf01BFa07B7a8Ee641690816d',

## Allowed Options

| Param | Nested | Type | Default | Remarks |
|----------|---------|---------|---------|---------|
| messageObj | - | json | - | message to be sent |
| - | content | string | message to be sent |
| - | meta? | META_MESSAGE_META | required for some message type |
| messageType | - | 'Text' &#124; 'Image' &#124; 'File' &#124; 'MediaEmbed' &#124; 'Meta' | 'Text'| type of messageContent |
| receiverAddress | - | string | - | user address or group chat id (Partial CAIP) |
| signer | - | - | - | signer object |
| pgpPrivateKey? | - | string | null | **mandatory for users having pgp keys** |
| env | - | string | 'prod' | API env - 'prod', 'staging', 'dev'|
